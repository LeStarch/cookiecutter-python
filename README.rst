cookiecutter-sphinx-docs
===================

A Cookiecutter_ template for Sphinx.

.. _cookiecutter: https://github.com/audreyr/cookiecutter


Usage
------

Install cookiecutter::

    $ pip install cookiecutter

Run it against this repo::

    $ cookiecutter
